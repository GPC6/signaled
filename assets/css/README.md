# CSS 작업 안내

현재 `style.css`는 1차 폴더의 메인화면 구도를 그대로 복원하기 위해 실제 홈페이지가 읽는 기준 CSS입니다.

앞으로 메뉴별 디자인을 많이 바꿀 때는 아래처럼 진행하는 것을 권장합니다.

1. 전체 공통 스타일: `/style.css`
2. 메인페이지 디자인: `/index.html` + `/style.css`의 Home/Hero 관련 영역
3. 각 메뉴별 세부 디자인: 해당 HTML에 section class를 추가한 뒤 `/style.css` 아래쪽에 모아서 작성

`assets/css` 안의 분리 CSS는 2차 정리 과정에서 만든 참고용 파일입니다. 현재 홈페이지 출력 안정성을 위해 기본 로딩은 `/style.css` 하나를 기준으로 맞췄습니다.
